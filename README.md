# Git Lab Assignment

## What I Learned
Through this assignment, I learned how to use Git as a version control system to manage project changes efficiently. I understood how to initialize a repository, track changes, and commit updates. I also learned how branching allows parallel development and how merging integrates different changes into a single codebase.

Additionally, I gained hands-on experience with pull requests and how they are used in collaborative workflows. I also learned how to handle merge conflicts manually, which improved my understanding of how Git manages conflicting changes.

## Challenges Faced
One of the main challenges I faced was understanding and resolving merge conflicts. Initially, the conflict markers were confusing, and I was unsure which changes to keep. However, after practicing, I understood how to manually edit and resolve conflicts properly.

Another challenge was remembering the sequence of Git commands and understanding when to use staging, committing, branching, and merging.

## How Git Helps in Real Projects
Git plays a critical role in real-world software development by allowing developers to track every change made to the codebase. It ensures that no work is lost and enables developers to revert to previous versions if necessary.

Git also supports collaboration by allowing multiple developers to work on different features simultaneously using branches. Changes can be reviewed and merged systematically using pull requests, improving code quality and reducing errors.

## Reflection
Version control systems like Git significantly improve software development quality and teamwork. By maintaining a structured history of all changes, Git allows developers to track progress, debug issues, and safely experiment with new features.

It enhances teamwork by enabling multiple developers to collaborate without overwriting each other's work. Features like branching, merging, and pull requests ensure that development is organized, efficient, and less prone to errors. Overall, Git is an essential tool for modern software development.
